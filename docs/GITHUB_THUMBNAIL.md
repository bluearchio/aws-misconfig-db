# Creating the GitHub Repository Thumbnail

## Setting the Social Preview Image

The repository thumbnail (social preview) appears when your repo is shared on social media and in GitHub search results.

### Image Specifications

- **Dimensions**: 1280x640 pixels (2:1 aspect ratio)
- **Format**: PNG or JPG
- **Max file size**: 1 MB
- **Recommended**: High contrast, clear text, professional design

### How to Set the Thumbnail

1. Go to your repository: https://github.com/bluearchio/aws-misconfig-db
2. Click **Settings** (repository settings, not user settings)
3. Scroll down to **Social preview**
4. Click **Edit**
5. Upload your image (1280x640px)
6. Click **Save**
